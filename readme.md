# Gobo Image Processor

This project is for accompanying interactive-layout plugins for MA2 OnPC. Requires a compatible Lua console plugin for use with showfile, which can be found [here](http://giaffodesigns.com/plugins/).

## Installation

No installation necessary on distributed binaries, as the project uses .NET Framework 4.7, which is already installed for any OnPC users.
